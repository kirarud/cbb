# Архитектура (кратко)

Последнее обновление: 2026-02-06

## Идея
Один общий стейт управляет UI, визуалом, аудио и логикой ИИ.
Интерфейс — это «кожа» внутреннего состояния.

## Слои
1. UI (React/Tailwind)
2. Сервисы (ai, synth, memory, physics)
3. Визуал (VisualCortex, Synesthesia, MatrixVision)
4. Персистентность (localStorage / IndexedDB)

## Ключевые сервисы
- geminiService.ts (также работает с Ollama)
- synthService.ts (генеративная музыка)
- muzaAIService.ts (гипербиты/физика)

## Важные правила
- Все строки UI — на русском
- Если нет optics — использовать безопасные фолбэки
- Никаких разрушительных действий без подтверждения
