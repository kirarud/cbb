# Принятые решения

Последнее обновление: 2026-02-06

## Продуктовые
- MVP — локальный и offline‑first.
- ИИ по умолчанию — Ollama (llama3.1:8b).
- Интерфейс — русский, без англ. текста там, где можно.
- Музыка не должна отключаться при переключении вкладок.
- VisualCortex стабилизируем управлением (пан/зум) и меньшим радиусом.

## Технические
- Безопасные фолбэки, если нет данных (optics / графа).
- Лёгкое персистентное состояние (localStorage + опционально IndexedDB).
- Модули независимы, но синхронизированы единым стейтом.
